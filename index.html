My last senior college project for a class that combines microcontroller protocols. A sensor hub that reads environment temperature (ADC) & humidity, and sending it's data via Bluetooth (UART).

The image on the left was the prototype wiring for the BlueTooth sensor hub.
The image on the right is the output coming from the microcontroller.

Schematic

The wiring in the schematic is how the microcontroller (Atmega2560) is connected to the respective peripherals.

The thermistor uses analog with a +5V reference.
The DHT22 sensor is connected to a digital pin, acting as both an input and an output.
The BlueTooth HC-06 module is connected to UART to connect to other devices, specifically to my cellphone.

Spanish Learning

A site created by me to help simplify the process of learning Spanish. It is a site intended for minimal explanations for how each tense operates while keeping the subject straight to the point. This is still a WIP.

Site Design

The site itself is divided into 3 sections. The navigation, content, and practice problems. All matter subjects are equally given these 3 sections for a uniformed layout, giving consistency to the overall site.

Practice Problems

Users are always presented practice problems at anytime to the far right, and can toggle which kind of problems they wish to practice with. There is no grading system as the main purpose here is to help the user learn Spanish, regardless of right or wrong answers.

The design I went for here was a very uniform layout, keeping the same code syntaxes consistent & similar across all pages. JavaScript functions are different for each tense, but are all written in the same format.

Videos & concepts below are what inspired me to push it to make a site version of what is shown below. Initially, I used Google Sheets & it's scripting language to randomly generate Spanish conjugation exercises. It has since been translated over to JavaScript, utilized as effective practice problems.
